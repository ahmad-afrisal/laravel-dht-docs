## 1. Tools / Aplikasi

### 1.1 Text Editor (VS Code / Cursor)

Visual Studio Code dan Cursor adalah editor teks modern yang ringan namun sangat powerful. VS Code sangat populer di kalangan developer karena fitur-fiturnya yang lengkap dan dukungan ekstensi yang luas, sementara Cursor menawarkan pengalaman coding dengan AI copilot terintegrasi.

### 1.2 Browser (Chrome)

Google Chrome digunakan untuk menjalankan dan menguji aplikasi web secara lokal. Tools developer-nya sangat membantu dalam proses debugging, inspeksi elemen, dan pemantauan jaringan.

### 1.3 Laragon / XAMPP

Laragon dan XAMPP adalah tool lokal server environment yang digunakan untuk menjalankan PHP dan MySQL secara offline. Laragon lebih ringan dan modern, cocok untuk development Laravel.

### 1.4 Design UI/UX (Figma)

Figma digunakan untuk membuat desain UI/UX secara kolaboratif. Sangat populer karena berbasis web, memungkinkan desain antar tim secara real-time tanpa instalasi aplikasi.

### 1.5 Git & GitHub

Git adalah version control system, sedangkan GitHub adalah platform hosting untuk menyimpan, mengelola, dan berbagi repository Git. Sangat penting untuk kolaborasi dan tracking versi aplikasi.

### 1.6 Node.js & NPM

Node.js memungkinkan kita untuk menjalankan JavaScript di luar browser, dan NPM digunakan sebagai package manager untuk menginstall library atau tools seperti Vite, Laravel Mix, dsb.

## 2. Extension Text Editor

### 2.1 Laravel Artisan

Menyediakan akses ke perintah artisan Laravel langsung dari dalam editor, mempercepat proses development tanpa perlu membuka terminal terpisah.

### 2.2 Laravel Extra Intellisense

Menambahkan fitur intellisense untuk Laravel di VS Code, seperti auto-complete route, model, view, dan helper, yang mempercepat penulisan kode.

### 2.3 Laravel Blade Snippets

Menyediakan shortcut/snippet untuk sintaks Blade Laravel agar penulisan template lebih cepat dan efisien.

### 2.4 Auto Close Tag

Secara otomatis menambahkan tag penutup untuk HTML/XML, mempercepat penulisan markup dan mencegah kesalahan.

### 2.5 Auto Rename Tag

Saat kamu mengganti nama tag pembuka, tag penutup akan ikut berubah secara otomatis. Sangat membantu untuk menjaga struktur HTML/Blade tetap rapi.

### 2.6 Database Client

Ekstensi untuk koneksi langsung ke database (MySQL, PostgreSQL, dll) dari dalam editor VS Code tanpa harus membuka software terpisah.

### 2.7 Database Client JDBC

Client JDBC memungkinkan koneksi ke berbagai database menggunakan driver JDBC, berguna untuk proyek yang memakai database enterprise seperti Oracle atau SQL Server.

### 2.8 Prettier - Code Formatter

Tool formatter populer yang secara otomatis merapikan dan menstandarisasi format kode agar lebih konsisten dan mudah dibaca.

### 2.9 Laravel Blade Snippets

Shortcut untuk Blade syntax, membantu mempercepat proses coding dengan template Laravel.

### 2.10 PHP Intelephense

Extension wajib bagi developer PHP. Menyediakan fitur auto-complete, go to definition, dan intellisense yang powerful untuk proyek PHP dan Laravel.

### 2.11 Laravel Goto View

Memungkinkan kita untuk langsung lompat ke file Blade terkait hanya dengan klik pada route/controller, mempercepat navigasi antar file Laravel.

### 2.12 Indent-rainbow

Memberikan warna berbeda untuk setiap level indentasi, membantu membaca struktur kode dengan lebih jelas terutama untuk HTML dan Blade.

### 2.13 Dracula Theme Official

Tema gelap yang populer di kalangan developer karena nyaman di mata, mendukung banyak bahasa, dan memiliki kontras warna yang baik.

### 2.14 Material Icon Theme

Mengganti ikon file di sidebar VS Code dengan ikon bergaya Material Design. Membantu membedakan tipe file secara visual.

### 2.15 GitHub Copilot

Asisten AI coding yang bisa menyarankan kode secara otomatis berdasarkan konteks, membantu mempercepat penulisan fungsi, logika, dan boilerplate.
