Anda hanya butuh 3 terminal ini saja:

Terminal 1: php artisan serve --host=0.0.0.0 (Untuk terima data ESP32).

Terminal 2: php artisan reverb:start --debug (Untuk mengalirkan data ke browser).

Terminal 3: npm run dev (Agar Laravel Echo di browser aktif).
