# Buat Akun EMQX Cloud

1. Kunjungi Halaman resmi EMQX CLOUD di https://www.emqx.com/en

![Deskripsi Gambar](/images/emqx-1.png)

2. Klik Tombol Start
![Deskripsi Gambar](/images/emqx-2.png)

3. Pilih Cloud
![Deskripsi Gambar](/images/emqx-3.png)

4. Pilih Serverless
![Deskripsi Gambar](/images/emqx-4.png)

5. Pilih Tipe Login
Sara Menggunakan Google agar mudh
![Deskripsi Gambar](/images/emqx-5.png)

6. Pilih Akun Google
![Deskripsi Gambar](/images/emqx-6.png)

7. Konfirmasi Akun Google. Pilih Continue
![Deskripsi Gambar](/images/emqx-7.png)

8. Isi Data Pribadi, Seperti First Name, Last Name, Company (Isi Random Saja), Country dan Phone Code, Dan Phone Number

![Deskripsi Gambar](/images/emqx-8.png)

9. Buat Project EMQX Baru

![Deskripsi Gambar](/images/emqx-9.png)

10. Pilih `+ New EMQX Broker`
![Deskripsi Gambar](/images/emqx-10.png)

11. Untuk Plan Pilih `serverless` dan Region pilih `North America`
![Deskripsi Gambar](/images/emqx-11.png)

12. Isi Deployment Name lalu klik deploy
![Deskripsi Gambar](/images/emqx-12.png)

13. Pilih Card EMQX Brokers
![Deskripsi Gambar](/images/emqx-13.png)

14. Pilih Access Control > Authentication
![Deskripsi Gambar](/images/emqx-14.png)

15. Buat Akun Untuk Perangkat IOT dengan tekan `+ Add`
![Deskripsi Gambar](/images/emqx-15.png)

16. Lalu isi Username dan Passwordnya
![Deskripsi Gambar](/images/emqx-16.png)

17. Buat Akun Untuk Laravel Serve `+ Add`
![Deskripsi Gambar](/images/emqx-17.png)

18. Lalu isi Username dan Passwordnya
![Deskripsi Gambar](/images/emqx-18.png)

19. Catat Host Address (contoh: x12345.emqxsl.com) dan pastikan port 8883 tersedia.
![Deskripsi Gambar](/images/emqx-19.png)

20. Unduh file sertifikat CA (`emqxsl-ca.crt`) 
![Deskripsi Gambar](/images/emqx-20.png)

21. letakkan di folder Laravel Anda: `storage/app/private/mqtt/emqxsl-ca.crt`.
![Deskripsi Gambar](/images/emqx-21.png)
